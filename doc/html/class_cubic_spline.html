Detailed Description

template<class T, typename R = float>
class CubicSpline< T, R >

CubicSpline Describes a CubicSpline defined by ControlPoints. Each ControlPoint consist of at least a position, and optionally a speed (tangent) and a time (these two can be set to default values, giving you a [0, 1] catmull-rom spline).
