Detailed Description

template<class T, typename R = float>
class CubicSpline< T, R >::ControlPoint

ControlPoint Describes a point in space (T) where an object following the spline should be at a given time (R), at a given speed (T). Speed and Time can be omitted, in this case, the CubicSpline class will use default values for there two properties.
